# InfluencerHub

B2B SaaS platform connecting influencers and brands with AI insights and immutable blockchain records.

## Tech Stack

- **Backend**: FastAPI
- **Database**: PostgreSQL (Supabase)
- **Auth**: Supabase Auth (JWT)
- **Frontend**: Streamlit
- **Blockchain**: Polygon Mumbai (Web3.py)
- **Payments**: Razorpay
